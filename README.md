# Bus Route Tracker

A 3D graphical application for tracking a bus route with interactive passenger management and inspection system.

## Project Description

This application simulates a bus traveling along a circular route with 10 stations. Users can interact with the bus system by adding/removing passengers and triggering random inspections.

## Controls

| Action           | Control                             |
| ---------------- | ----------------------------------- |
| Add Passenger    | Left Mouse Click (doors open only)  |
| Remove Passenger | Right Mouse Click (doors open only) |
| Send Inspector   | `K` Key (doors open only)           |
